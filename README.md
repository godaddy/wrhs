# Usage
<!-- usage -->
```sh-session
$ npm install -g wrhs
$ wrhs COMMAND
running command...
$ wrhs (-v|--version|version)
wrhs/1.0.0 darwin-x64 node-v14.15.0
$ wrhs --help [COMMAND]
USAGE
  $ wrhs COMMAND
...
```
